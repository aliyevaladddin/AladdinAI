# Security Policy

## Supported Versions

The following versions of AladdinAI are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of AladdinAI and the Sovereign Tech ecosystem very seriously. If you discover a security vulnerability, please do NOT open a public issue. Instead, follow these steps:

1. **Private Report:** Send a detailed report to [INSERT YOUR SECURITY EMAIL] describing the vulnerability.
2. **Details:** Include a proof-of-concept or steps to reproduce the issue.
3. **Response:** You will receive an acknowledgment of your report within 48 hours.
4. **Resolution:** We will work to resolve the issue as quickly as possible and will keep you informed of our progress.

## Our Commitment

We are committed to providing a secure environment for all users and contributors. As a Sovereign Tech project, we prioritize data privacy and cryptographic integrity above all else.
